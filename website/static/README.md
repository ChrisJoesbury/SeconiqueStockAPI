# static/

This directory contains the project's **source** static files — the CSS, JavaScript, and images written for this project.

It is referenced in `settings.py` via `STATICFILES_DIRS` and is committed to the repository. **Tested on:** Django 6.0.1.

> **Note:** This is distinct from `staticfiles/` (the `STATIC_ROOT`), which is auto-generated by `python manage.py collectstatic` and is **not** committed to the repository.

---

## 📁 Contents

```
static/
├── css/
│   └── templateStyles.css    # Global styles for all HTML templates
├── images/
│   └── seconique-logo.png    # Seconique brand logo used in templates
└── js/
    └── csrf.js               # CSRF token helper for JavaScript fetch requests
```

---

## 📝 File Descriptions

### `css/templateStyles.css`
Shared stylesheet applied across the login, register, profile, and CSV download pages.

### `images/seconique-logo.png`
The Seconique logo displayed in the site header and login page.

### `js/csrf.js`
A JavaScript utility that attaches the Django CSRF token to outgoing `fetch` requests. Required for any JS that makes POST requests to the Django backend (e.g. the CSV download page).
